Когда сервер запущен, можете выбрать:

1. Список устройств
   → Показывает все подключённые телефоны
   
2. Переключить камеру
   → Переключение передней/задней на выбранном телефоне
   
3. Сделать фото
   → Фото сохраняется в photos/ на ПК
   
4. Broadcast сообщение
   → Отправляет сообщение на ВСЕ телефоны
   
5. Выход
   → Закрывает сервер

┌────────────────────────────────────────────────────────────────┐
│ ❌ ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ                                  │
└────────────────────────────────────────────────────────────────┘

ОШИБКА: Connection refused
   → Сервер не запущен / неправильный IP / разная сеть

ОШИБКА: Python не найден
   → Переустановите Python (отметьте Add to PATH)

ОШИБКА: ModuleNotFoundError: kivy
   → pip install kivy --upgrade

ОШИБКА: Приложение не устанавливается
   → На телефоне разрешите неизвестные источники

ОШИБКА: Камера не работает
   → На телефоне разрешите доступ к камере
