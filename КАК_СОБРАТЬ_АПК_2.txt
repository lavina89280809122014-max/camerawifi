🥇 СПОСОБ 1: GitHub Actions (САМЫЙ ПРОСТОЙ) ⭐
═══════════════════════════════════════════════════════════════════════════════

Бесплатно через GitHub - всё автоматически!

ШАГ 1: Создайте GitHub аккаунт (если нет)
   https://github.com

ШАГ 2: Загрузите проект на GitHub
   1. Создайте новый репозиторий
   2. Загрузите все файлы из папки
   3. Коммит и push

ШАГ 3: Создайте GitHub Actions workflow
   1. Создайте файл: .github/workflows/build.yml
   
   Содержимое:
   ───────────────────────────────────────────────────────────
   name: Build APK
   on: [push]
   jobs:
     build:
       runs-on: ubuntu-latest
       steps:
         - uses: actions/checkout@v2
         - name: Setup Python
           uses: actions/setup-python@v2
           with:
             python-version: '3.9'
         - name: Install dependencies
           run: |
             pip install buildozer cython
         - name: Build APK
           run: |
             buildozer -f -v android debug
         - name: Upload APK
           uses: actions/upload-artifact@v2
           with:
             name: APK
             path: bin/
   ───────────────────────────────────────────────────────────

ШАГ 4: Push файл в репозиторий
   GitHub автоматически соберёт APK!

ШАГ 5: Скачайте APK
   Откройте Actions → найдите build → скачайте artifact

✅ ГОТОВО! APK собран! 🎉

🥉 СПОСОБ 3: Используйте готовый скрипт (ПРОСТОЙ)
═══════════════════════════════════════════════════════════════════════════════

Я создам вам BAT скрипт для автоматической сборки:


АВТОСБОРКА.bat
───────────────────────────────────────────────────────────
@echo off
REM Автоматическая сборка APK

echo.
echo ============================================
echo   APK АВТОМАТИЧЕСКАЯ СБОРКА
echo ============================================
echo.

REM Проверка Python
python --version >nul 2>&1
if errorlevel 1 (
    echo ошибка: Python не установлен
    pause
    exit /b 1
)

REM Установка зависимостей
echo Устанавливаю зависимости...
pip install buildozer cython

REM Сборка
echo.
echo Начинаю сборку APK...
echo Это займёт 20-30 минут...
echo.

cd /d "%~dp0"
buildozer -f -v android debug

if errorlevel 1 (
    echo.
    echo ОШИБКА при сборке!
    pause
    exit /b 1
)

echo.
echo ============================================
echo   ✅ СБОРКА ЗАВЕРШЕНА УСПЕШНО!
echo   APK находится в: bin/camerawifi-1.0-debug.apk
echo ============================================
echo.
pause
───────────────────────────────────────────────────────────

Просто сохраните как АВТОСБОРКА.bat в папку проекта
и дважды клик для запуска!
