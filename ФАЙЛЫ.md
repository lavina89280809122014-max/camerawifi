# 📱 ПРОЕКТ: КАМЕРА WIFI - СИСТЕМА УДАЛЁННОГО УПРАВЛЕНИЯ КАМЕРОЙ

## 🎯 ЧТО ЭТО?

Полная система для управления камерой телефона с компьютера через WiFi:

```
КОМПЬЮТЕР (Windows/Mac/Linux)        ТЕЛЕФОН (Android)
      ↓                                    ↓
  server.py ←──── WiFi ────→ camerawifi.apk
```

- Запустите сервер на компьютере
- Установите приложение на телефон
- Управляйте камерой удаленно с компьютера
- Получайте уведомления в реальном времени

## 📡 КАК ЭТО РАБОТАЕТ

### Архитектура

1. **Сервер (компьютер)**
   - Слушает порт 5555
   - Принимает подключения телефонов
   - Отправляет команды
   - Выводит статус в интерактивное меню

2. **Клиент (телефон)**
   - Подключается к серверу
   - Выполняет команды (переключение камеры, фото)
   - Отправляет уведомления

3. **Сообщения (JSON)**
   - Все данные в формате JSON
   - Простые, человеко-читаемые сообщения

### Порядок работы

```
Телефон                          Компьютер
  ↓                                 ↓
Запуск приложения
  ↓
Ввод IP и порта
  ↓
Нажимаю "Подключить"
  ├─ Отправляю: device_info ────→ Сервер получает
  ↓                              ├─ Выводит уведомление
Получаю: "✅ Подключено"  ←────── Сервер подтверждает
  ↓                                 ↓
Ожидаю команд              Меню: выбираю действие
  ↓                                 ↓
  ←────────── Команда command ────  Отправляю
  ↓
Выполняю команду
(переключение камеры / фото)
  ├─ Отправляю: photo_saved ────→ Сервер выводит статус
  ↓
Получаю уведомление
```

## 🎮 УПРАВЛЕНИЕ

### На компьютере (server.py)

```
1. Список устройств
   → Показывает все подключённые телефоны
   → Каждому присваивается номер

2. Переключить камеру
   → Выбираете номер телефона
   → На телефоне переключается передняя/задняя камера

3. Сделать фото
   → Выбираете номер телефона
   → На телефоне делается снимок
   → Фото сохраняется в папку photos/

4. Broadcast сообщение
   → Пишете текст сообщения
   → Сообщение отправляется на ВСЕ подключённые телефоны
   → Появляется уведомление на телефонах

5. Выход
   → Завершает сервер
```

### На телефоне (camerawifi.apk)

```
🌐 Подключение
├─ IP адрес:        192.168.1.100 (ввод)
├─ Порт:             5555 (по умолчанию)
├─ ✅ Подключить:   Подключиться к серверу
└─ ❌ Отключить:    Разорвать соединение

📷 Камера
├─ 🔄 Переключить:  Переключение передней/задней
├─ 📸 Фото:         Сохранение снимка
└─ ✕ Выход:         Закрыть приложение

📊 Статус
├─ 🟢 Подключено:    Соединение активно
└─ 🔴 Отключено:     Нет соединения
```

### Частые ошибки

| Ошибка | Решение |
|--------|---------|
| ModuleNotFoundError: kivy | pip install kivy --upgrade |
| Connection refused | Проверьте IP и порт, сервер запущен? |
| Permission denied | Разрешите доступ к камере на телефоне |
| Network unreachable | Телефон и ПК в одной WiFi сети? |

## 📊 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Сценарий 1: Фотосъёмка на расстоянии

```
Друг держит телефон, вы управляете с ноутбука
1. Телефон подключён к серверу
2. На компьютере: меню → 3 (Фото)
3. Фото делается автоматически
4. Результат сохраняется
```

### Сценарий 2: Переключение камер

```
Нужно переключиться с задней на переднюю
1. На компьютере: меню → 2 (Переключить)
2. На телефоне переключится камера
3. На компьютере можно переключать много раз
```

### Сценарий 3: Несколько телефонов

```
Управление разными устройствами
1. Подключите 3 телефона
2. На компьютере: меню → 1 (Список устройств)
   └─ 1. 192.168.1.50:54321
   └─ 2. 192.168.1.51:54322
   └─ 3. 192.168.1.52:54323
3. На компьютере: меню → 2 → выбираете номер
4. Команда отправляется выбранному устройству
```
