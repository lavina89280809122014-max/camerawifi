2️⃣ НА ТЕЛЕФОНЕ

   • Установите camerawifi-1.0-debug.apk
   • Откройте приложение "📷 Камера WiFi"
   • Введите IP из шага 1️⃣ (например: 192.168.1.100)
   • Порт оставьте 5555
   • Нажмите ✅ Подключить

   На телефоне появится:

   На компьютере появится:
   ✅ Новое подключение: 192.168.1.50:54321
   📱 Всего устройств подключено: 1

🎮 ПРИМЕРЫ КОМАНД
───────────────────────────────────────────────────────────────

ПРИМЕР 1: Переключить камеру
─────────────────────────────
Компьютер → Меню → 2 → Выбираете телефон → Отправка

На компьютере выведет:
📤 Команда отправлена на 192.168.1.50:54321: switch_camera

На телефоне:
• Камера переключится
• Камера отправит уведомление серверу

ПРИМЕР 2: Сделать фото
──────────────────────
Компьютер → Меню → 3 → Выбираете телефон → Отправка

На компьютере выведет:
📤 Команда отправлена на 192.168.1.50:54321: take_photo

На телефоне:
• Сделается фотография
• Фото сохранится в photos/photo_TIMESTAMP.png
• Отправит уведомление серверу

На компьютере:
📸 Фото сохранено: photos/photo_1234567890.png

ПРИМЕР 3: Сообщение всем телефонам
────────────────────────────────────
Компьютер → Меню → 4 → Пишете текст → Отправка

На компьютере:
Введите сообщение: Привет всем!

На ВСЕ телефонах появится уведомление:
🔔 Уведомление
Привет всем!

🐛 РЕШЕНИЕ ПРОБЛЕМ
───────────────────────────────────────────────────────────────

ПРОБЛЕМА: "Connection refused"
ПРИЧИНА: Сервер не запущен или неправильный IP адрес
РЕШЕНИЕ: 
  1. Проверьте что сервер запущен
  2. Проверьте IP адрес (должен быть как на компьютере)
  3. Проверьте что телефон и ПК в одной WiFi сети

ПРОБЛЕМА: "Python не найден"
ПРИЧИНА: Python не установлен или не в PATH
РЕШЕНИЕ:
  1. Установите Python с https://www.python.org/downloads/
  2. ✓ Отметьте "Add Python to PATH"
  3. Перезагрузите компьютер

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'kivy'"
ПРИЧИНА: Kivy не установлен
РЕШЕНИЕ:
  pip install kivy --upgrade

ПРОБЛЕМА: "Приложение не устанавливается на телефон"
ПРИЧИНА: Неизвестные источники не разрешены
РЕШЕНИЕ:
  1. На телефоне: Параметры → Безопасность
  2. Включить "Установка из неизвестных источников"

ПРОБЛЕМА: "Камера не работает на телефоне"
ПРИЧИНА: Приложению не разрешен доступ к камере
РЕШЕНИЕ:
  1. На телефоне: Параметры → Приложения
  2. Выберите "Камера WiFi"
  3. Разрешить доступ к Камере

ПРОБЛЕМА: "Соединение очень медленное"
ПРИЧИНА: Плохое качество WiFi
РЕШЕНИЕ:
  1. Подойдите ближе к маршрутизатору
  2. Проверьте скорость WiFi
  3. Закройте другие приложения

⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ:

✅ ИСПОЛЬЗУЙТЕ:
  • В локальной сети (дома, офис)
  • С надежным WiFi
  • Только доверенные устройства

❌ НЕ ИСПОЛЬЗУЙТЕ:
  • Публичный WiFi
  • Интернет напрямую (без VPN)
  • Неизвестные сети

🔐 РЕКОМЕНДАЦИИ:
  • Используйте сложный пароль WiFi
  • Регулярно обновляйте Python и Kivy
  • Отключайте приложение когда не используете
  • Не делитесь IP адресом
