## 🔧 УСТАНОВКА НА КОМПЬЮТЕР

### Шаг 1: Установка Python (если не установлен)

1. Скачайте Python 3.9+ с https://www.python.org/downloads/
2. ⚠️ При установке отметьте "Add Python to PATH"

### Шаг 2: Установка зависимостей

Откройте командную строку/PowerShell и выполните:

```bash
# Навигация в папку проекта
cd "d:\Новая папка (3)"

# Установка необходимых пакетов
pip install kivy
```

## 🚀 ЗАПУСК НА КОМПЬЮТЕРЕ

### 1. Узнайте IP адрес компьютера

**На Windows:**
```bash
ipconfig
```
Найдите строку `IPv4 Address` (обычно вроде `192.168.1.100`)

**На Mac/Linux:**
```bash
ifconfig
```

### 2. Запустите сервер

```bash
cd "d:\Новая папка (3)"
python server.py
```

**Вывод должен быть:**
```
==================================================
🖥️  СЕРВЕР КАМЕР - КОМПЬЮТЕРНАЯ ЧАСТЬ
==================================================
IP адрес: 192.168.1.100
Порт: 5555
Адрес для подключения: 192.168.1.100:5555
==================================================

🔧 Сервер запущен на 0.0.0.0:5555
📡 Ожидание подключений...
```

### 3. Интерактивное меню на компьютере

Когда сервер запущен, вы можете использовать меню:

```
==================================================
🎮 КОМАНДЫ УПРАВЛЕНИЯ
==================================================
1. Список устройств      - показывает подключённые телефоны
2. Переключить камеру    - отправляет команду переключить камеру
3. Сделать фото         - отправляет команду сделать фото
4. Broadcast сообщение  - отправляет сообщение всем устройствам
5. Выход                - выключает сервер
==================================================
```

## 📱 УСТАНОВКА НА ТЕЛЕФОН

### Способ 1: Через APK файл (РЕКОМЕНДУЕТСЯ)

#### Шаг 1: Сборка APK

На компьютере выполните:

```bash
cd "d:\Новая папка (3)"

# Установка buildozer и зависимостей
pip install buildozer cython

# Сборка APK (первый раз будет долго - 15-30 минут)
buildozer -f -v android debug
```

**Результат:** `bin/camerawifi-1.0-debug.apk`

#### Шаг 2: Установка на телефон

**Способ 1 - Через ADB (Android Debug Bridge):**

1. Установите Android Platform Tools:
   - Скачайте с https://developer.android.com/tools/releases/platform-tools
   - Распакуйте в удобное место

2. Подключите телефон USB кабелем

3. На телефоне включите "Режим разработчика":
   - Параметры → О телефоне → 7 раз нажмите на "Номер сборки"

4. В командной строке выполните:
```bash
cd path\to\platform-tools
adb install "d:\Новая папка (3)\bin\camerawifi-1.0-debug.apk"
```

**Способ 2 - Через файловый менеджер:**

1. Подключите телефон USB кабелем
2. Скопируйте `camerawifi-1.0-debug.apk` в память телефона
3. На телефоне откройте файловый менеджер
4. Найдите и откройте APK файл
5. Нажмите "Установить"

**Способ 3 - Отправка по WiFi:**

1. На компьютере включите общий доступ к папке или используйте облако
2. На телефоне скачайте APK
3. Установите как описано выше

### Способ 2: Запуск напрямую (для тестирования)

На компьютере:

```bash
pip install kivy

# Запуск приложения на компьютере (имитация телефона)
python client.py
```

## 🔌 ИСПОЛЬЗОВАНИЕ

### На компьютере (сервер)

1. Запустите `server.py`
2. Убедитесь, что сервер ожидает подключений
3. Используйте меню для управления устройствами:
   - Введите `1` → список подключённых устройств
   - Введите `2` → выберите устройство и отправьте команду переключения камеры
   - Введите `3` → выберите устройство и отправьте команду фото

### На телефоне (приложение)

1. Запустите приложение "Камера WiFi"
2. Введите IP адрес компьютера (например: `192.168.1.100`)
3. Порт: `5555` (по умолчанию)
4. Нажмите ✅ Подключить
5. На компьютере увидите:
   ```
   ✅ Новое подключение: 192.168.1.50:54321
   📱 Всего устройств подключено: 1
   ```
6. На телефоне появится уведомление ✅ Подключено к серверу
7. Используйте приложение:
   - 🔄 Переключить - переключение передней/задней камеры
   - 📸 Фото - сохранение снимка
   - ❌ Отключить - разорвать соединение

## 🐛 РЕШЕНИЕ ПРОБЛЕМ

### "Соединение отказано"

**Причины:**
- Сервер не запущен на компьютере
- Неправильный IP адрес
- Телефон и компьютер в разных сетях
- Брандмауэр блокирует порт 5555

**Решение:**
```bash
# На Windows добавьте исключение в брандмауэр
# Параметры → Брандмауэр → Разрешить приложение
# Выберите python.exe и разрешите

# Или отключите брандмауэр для теста
```

### "Модуль kivy не найден"

```bash
pip install kivy --upgrade
```

### "Приложение тормозит на телефоне"

- Проверьте качество WiFi соединения
- Закройте другие приложения
- Перезагрузите телефон

### "Камера не работает"

- На телефоне разрешите доступ к камере в параметрах
- Проверьте права доступа в приложении

## 🎯 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Пример 1: Удалённая фотосъёмка

1. На компьютере: меню → 3 (Фото)
