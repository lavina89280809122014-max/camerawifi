⚡ БЫСТРАЯ ДЕМОНСТРАЦИЯ (как это работает):
═══════════════════════════════════════════════════════════════════════════════

1. На компьютере запускаете: python server.py

   Вывод:
   ╔════════════════════════════════════════════╗
   ║ 🖥️  СЕРВЕР КАМЕР                         ║
   ║ IP адрес: 192.168.1.100                  ║
   ║ Порт: 5555                               ║
   ╚════════════════════════════════════════════╝

2. На телефоне вводите IP и подключаетесь

3. На компьютере появляется:
   ✅ Новое подключение: 192.168.1.50:54321
   📱 Всего устройств подключено: 1

4. На телефоне появляется:
   🟢 Статус: Подключено

5. На компьютере выбираете меню:
   🎮 КОМАНДЫ УПРАВЛЕНИЯ
   1. Список устройств
   2. Переключить камеру
   3. Сделать фото
   4. Broadcast сообщение
   5. Выход

6. Например, нажимаете "3" (Сделать фото)
   → На телефоне делается фото
   → На компьютере выводится: 📸 Фото сохранено!

───────────────────────────────

СЦЕНАРИЙ 3: Несколько телефонов
────────────────────────────────
   Управление 3 телефонами от одного ПК:
   
   На ПК: Меню → 1 → Список из 3 устройств
          Меню → 2 → Выбираете устройство 2
          Команда отправляется только на телефон 2 ✅
